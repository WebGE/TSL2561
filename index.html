TSL2561 LIGHT-TO-DIGITAL CONVERTER

Le capteur TSL2561 permet de mesurer une luminosité de 0,1 à 40000 Lux. Il communique avec un microcontrôleur via le bus I2C.

Son capteur couvre la lumière visible et les infrarouges grâce à deux diodes. Cela signifie que vous pouvez mesurer séparément la lumière infrarouge, la lumière visible ou le spectre complet.

  • Alimentation: 2,7 à 3,6 Vcc
  • Interface: I2C
  • Plage de mesure: 0,1 à 40000 Lux
  • Température de service: -30 à +80 °C
  • Documentation: TSL2561 LIGHT-TO-DIGITAL CONVERTER
  • Distributeur: Gotronic

Classe TSL2561

Contenu du dossier à télécharger

Le dossier à télécharger (au format zip ou tar.gz) contient le répertoire d'un projet Microsoft Visual Studio Professionnel composé :

  • Du code de la classe TSL2561.
  • D'un exemple d'utilisation de cette classe.

Les fichiers program.cs et DriversTSL2561.cs pourront être recompilés avec Microsoft Visual Studio Express.

Utilisation

  1. Créer un objet en utilisant le constructeur TSL2561 approprié.
  2. Initialiser le circuit avec la méthode Init et les paramètres appropriés.
  3. Lire la luminosité ambiante avec la méthode CalculateLux

Constructeurs

Syntaxe Description
TSL2561() Instancie un objet "TSL2561".
Son addresse sur le bus I2C = 0x29.
La fréquence du signal d'horloge est fixée à F = 100kHz.
TSL2561(byte I2C_Add_7bits) Instancie un objet "TSL2561".
I2C_Add_7bits est à définir en fonction de l'état de la broche d'adresse (GND)0x29, (VDD)0x49, (FLOAT)0x69.
La fréquence du signal d'horloge est fixée à F = 100kHz.
TSL2561(byte I2C_Add_7bits, UInt16 Frequency) Instancie un objet "TSL2561".
I2C_Add_7bits est à définir en fonction de l'état de la broche d'adresse (GND)0x29, (VDD)0x49, (FLOAT)0x69.
Frequency est à définir dans l'intervalle [100kHz,400kHz].

Enumérations

Syntaxe Description
Gain x1 : Fixe le gain à 1 (par défaut)
x16 : Fixe le gain à 16
IntegrationTime _13MS : 13,7ms
_101MS : 101ms
_402MS : 402ms (par défaut)

Propriétés

Syntaxe Description
Channel0 Retourne un mot de 16 bits non signé représentatif de la lumière ambiante.
Channel1 Retourne un mot de 16 bits non signé représentatif des infrarouges.
Control Retourne la valeur contenue dans le registre "CONTROL". Octet non signé.
id Retourne la valeur contenue dans le registre "ID". Octet non signé.
- quartet de poids fort : (0)TSL2560 (1)TSL2561
- quartet de poids faible : valeur de l'identifiant du composant
interrupt Retourne une valeur représentative du réglage des interruptions. Octet non signé.
threshhighhigh Retourne ou fixe la valeur maximum du seuil haut de luminosité. Octet non signé.
threshhighLow Retourne ou fixe la valeur minimum du seuil haut de luminosité. Octet non signé.
threshLowhigh Retourne ou fixe la valeur maximum du seuil bas de luminosité. Octet non signé.
threshLowLow Retourne ou fixe la valeur minimum du seuil bas de luminosité. Octet non signé.
timing Retourne la valeur contenue dans le registre "TIMING". Octet non signé.
